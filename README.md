# 📊 Amazon Stock Analysis Dashboard

## 📌 Project Overview

An interactive **Microsoft Excel Dashboard** designed to analyze Amazon's historical stock market performance. The dashboard provides insights into stock price movements, trading volume, daily trends, monthly averages, and yearly performance using dynamic charts, KPIs, and slicers.

---

# 🎯 Objective

* Monitor Amazon stock price performance over time.
* Compare Open, High, Low, and Close prices.
* Analyze trading volume trends.
* Identify monthly and yearly stock price patterns.
* Explore stock data interactively using filters.

---

# 📊 Dashboard Features

* Interactive Day, Month, and Year Slicers
* Dynamic KPI Cards
* Open vs High Price Comparison
* High vs Low Price Analysis
* Trading Volume Analysis
* Monthly Average Adjusted Close Price
* Yearly Adjusted Close Price Trend
* Day-wise Value Distribution
* Fully Interactive Visualizations

---

# 📈 Key Performance Indicators (KPIs)

* 📌 Average Open Price
* 📌 Average High Price
* 📌 Average Low Price
* 📌 Average Close Price
* 📊 Trading Volume
* 📅 Monthly Adjusted Close Price
* 📈 Yearly Stock Performance
